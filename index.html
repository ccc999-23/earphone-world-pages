TikTok Content Posting Tool
for Music Curators

Earphone World JP is a content management tool that enables music curators and creators to publish photo slideshow posts to TikTok using the TikTok Content Posting API.

The tool automates the creation and delivery of music recommendation content — including album artwork, lyrics cards, and curated captions — to TikTok via the official API.

Features

Photo Slideshow Publishing

Publish multi-image slideshow posts to TikTok. Each slide is generated from album artwork and lyrics, formatted for TikTok's photo format.

Music Metadata Integration

Automatically fetches album artwork and track information from music platforms to build visually consistent content.

Caption Generation

Generates platform-optimized captions and hashtags tailored for music recommendation content on TikTok.

Inbox Draft Support

Posts are sent to TikTok's inbox as drafts, allowing creators to review and edit before publishing.

API Usage

This application uses the TikTok Content Posting API with the following OAuth scopes:

user.info.basic — Reads the authenticated user's display name and profile information to confirm account identity.

video.upload — Uploads photo slideshow content to the authenticated user's TikTok inbox as a draft post.

No user data obtained via TikTok APIs is stored on external servers or shared with third parties. All API interactions occur on behalf of the authenticated user only.
